# SpringBoot

Use below annotations for understanding.

In AppConfig class :
1) @Configuration
2) @Bean
3) @Order
4) @Lazy

In UserController Class : 
1) @RestController
2) @Autowire
3) Setter Injection
4) @GetMapping

In Repository :
1) @Repository

In UserService Class : 
1) @Service
2) @Value
3) @PostConstruct
4) Construction Injection
